WebSQL
======

>
NodeJS 搭建静态服务器 + WebSql Database数据存储 + BackboneJS + SeaJS


## 说明  ##

1. WebSql Database作为数据存储，需在支持WebSql Database的浏览器中运行(google chrome)

2. SeaJS作为JS模块加载器

3. BackboneJS 前端MVC开发模式

4. NodeJS作为后端的静态资源服务器

*****
例子中运用到模版加载tpl，采用NodeJS作为后端服务器，实现全应用JS化开发

前后端JS学习实例
